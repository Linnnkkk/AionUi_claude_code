## 快速开始

### 环境要求

- Docker 20.10+
- Docker Compose 2.0+
- 至少 4GB 内存
- 至少 5GB 可用磁盘空间

### 快速部署

```bash
cd /path/to/coder-server-aionui

CODE_SERVER_HOME=/path/to/workspace \
CODE_SERVER_PASSWORD=your_secure_password \
./docker-setup.sh
```

**环境变量说明：**

| 变量 | 必需 | 说明 | 默认值 |
|------|------|------|--------|
| `CODE_SERVER_HOME` | ✅ | 工作目录（所有数据存储在此） | 无 |
| `CODE_SERVER_PASSWORD` | ✅ | Code-Server 密码 | 无 |
| `CODE_SERVER_PORT` | ❌ | Code-Server 外部端口 | `8080` |
| `AIONUI_PORT` | ❌ | AionUi 外部端口 | `25808` |
| `AIONUI_USERNAME` | ❌ | AionUi 用户名 | `admin` |
| `AIONUI_PASSWORD` | ❌ | AionUi 密码 | 首次随机生成 |
| `AIONUI_VERSION` | ❌ | AionUi 版本 | `1.7.8` |
| `NOVNC_PORT` | ❌ | noVNC 端口 | `6080` |
| `COMPOSE_PROJECT_NAME` | ❌ | 项目名称 | `openclaw` |
| `CONTAINER_NAME` | ❌ | 容器名称 | `openclaw` |

**注意：** Chrome 数据目录已移除，现在统一存储在 `CODE_SERVER_HOME/.chrome` 下。

### 访问地址

| 服务 | 地址 | 用途 |
|------|------|------|
| **Code-Server** | `http://localhost:<CODE_SERVER_PORT>` | 文件编辑、代码调试 |
| **AionUi** | `http://localhost:<AIONUI_PORT>` | AI 智能体 Web 界面 |
| **noVNC** | `http://localhost:<NOVNC_PORT>` | 浏览器可视化（按需） |

**密码说明：**
- **Code-Server**: 必须通过 `CODE_SERVER_PASSWORD` 设置
- **AionUi**: 可通过 `AIONUI_PASSWORD` 环境变量预设密码，未设置则首次登录随机生成

### 进入容器使用 CLI

```bash

docker exec -it <CONTAINER_NAME> bash

# 查看服务状态

```

## Chrome 管理指南

### Chrome Manager 命令

```bash
# 查看 Chrome 状态
chrome-manager status

# 启动 Chrome (headless - 纯命令行模式)
chrome-manager start-headless

# 启动 Chrome + VNC (可视化模式)
chrome-manager start-vnc

# 停止 Chrome 和 VNC
chrome-manager stop

# 重启 Chrome
chrome-manager restart

# 获取 CDP 地址
chrome-manager url
# 输出: http://localhost:9222
```

### 使用场景

#### 场景1：纯自动化任务（headless）

```bash
# 启动 headless Chrome
chrome-manager start-headless

# 智能体通过 CDP 控制
# 任务：爬取网页、填写表单、下载文件等

# 任务完成，停止 Chrome
chrome-manager stop
```

#### 场景2：需要可视化/调试

```bash
# 启动 Chrome + VNC
chrome-manager start-vnc

# 浏览器访问 noVNC
# http://localhost:<NOVNC_PORT>

# 在 noVNC 中查看浏览器界面
# 手动处理验证码、登录等
```

#### 场景3：混合模式（推荐）

```bash
# 默认使用 headless
chrome-manager start-headless

# 遇到验证码，切换到 VNC 模式
chrome-manager start-vnc
# 浏览器访问 http://localhost:<NOVNC_PORT> 手动处理

# 处理完成，继续自动化
```

### 手动登录账号

```bash
# 1. 启动 VNC 模式
chrome-manager start-vnc

# 2. 浏览器访问 noVNC
# http://localhost:<NOVNC_PORT>

# 3. 在 VNC 中打开 Chrome，登录账号

# 4. 登录成功后，Cookie 保存在 /home/coder/.chrome

# 5. 后续智能体可以通过 CDP 使用已登录的账号
```

## 常见问题

### Q1: Chrome 无法启动？

**检查状态：**
```bash
chrome-manager status
```

**常见原因：**
- 端口冲突（CDP 9222）
- 数据目录权限问题
- 内存不足

**解决方法：**
```bash
# 修复权限
sudo chown -R 1000:1000 $CODE_SERVER_HOME/.chrome

# 重启 Chrome
chrome-manager stop
chrome-manager start-headless
```

### Q2: noVNC 无法连接？

**检查端口：**
```bash
docker port <CONTAINER_NAME>
```

**预期输出：**
```
8080/tcp -> 0.0.0.0:8080
25808/tcp -> 0.0.0.0:25808
6080/tcp -> 0.0.0.0:6080  # noVNC
```

### Q3: Playwright 浏览器下载失败？

**Dockerfile 已配置多镜像源自动重试：**
1. npmmirror 淘宝镜像
2. npmmirror CDN
3. Playwright Azure CDN
4. Playwright Azure Akamai

**手动安装：**
```bash
# 进入容器
docker exec -it <CONTAINER_NAME> bash

# 设置镜像
export PLAYWRIGHT_DOWNLOAD_HOST=https://npmmirror.com/mirrors/playwright/

# 安装浏览器
playwright install chromium --with-deps
```

## 避坑指南

### 1. Bun 下载 404

**问题：** npmmirror 路径格式错误

**解决：** 使用 `bun-v${VERSION}` 前缀
```dockerfile
# 正确
curl -fsSL https://registry.npmmirror.com/-/binary/bun/bun-v${BUN_VERSION}/bun-linux-x64.zip

# 错误
curl -fsSL https://registry.npmmirror.com/-/binary/bun/${BUN_VERSION}/bun-linux-x64.zip
```

### 2. corepack: not found

**问题：** Node.js 版本不包含 corepack

**解决：** 重新安装 Node.js 22.x，再 corepack enable
```dockerfile
# 正确顺序
RUN curl -fsSL https://registry.npmmirror.com/-/binary/node/v22.11.0/node-v22.11.0-linux-x64.tar.xz | tar -xJC /usr/local --strip-components=1

RUN corepack enable  # 必须在重新安装 Node.js 之后
```

### 3. Playwright 浏览器下载

**v4.3 解决：** 使用官方安装命令 + 国内镜像
```dockerfile
# 设置镜像源
ENV PLAYWRIGHT_DOWNLOAD_HOST=https://npmmirror.com/mirrors/playwright/

# 安装浏览器（支持多镜像源重试）
RUN for MIRROR in \
    "https://npmmirror.com/mirrors/playwright" \
    "https://cdn.npmmirror.com/binaries/playwright" \
    "https://playwright.azureedge.net"; do \
    PLAYWRIGHT_DOWNLOAD_HOST="$MIRROR" playwright install chromium --with-deps && exit 0; \
done
```

### 4. AionUi 端口映射错误

**问题：** 端口映射 `${AIONUI_PORT}:${AIONUI_PORT}` 导致内外端口不一致

**解决：** 内部固定端口 25808
```yaml
# 正确
- "${AIONUI_PORT:-25808}:25808"
```
